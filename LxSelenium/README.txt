You can build the LxSelenium site documentation via this maven command:

    $ mvn site

Then point your web-browser at this file:

    LxSelenium/target/site/index.html

Alternatively you can view the docs as built by Bamboo at:

    http://bamboo301.lucernex.com:8085/browse/QA-LXSEL/latest/artifact/JOB1/Site-Docs/index.html
